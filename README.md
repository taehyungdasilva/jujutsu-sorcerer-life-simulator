
# Jujutsu Sorcerer Life Simulator

A text-based life simulation RPG inspired by:

* Jujutsu Kaisen
* The Fenomeno
* Football Manager
* Persona
* Crusader Kings
* RimWorld

## Vision

Create the definitive Jujutsu Sorcerer life simulator where players live the life of an original sorcerer, manage their growth, build relationships, survive major incidents, and leave a lasting legacy on the world.

## Current Status

Prototype v0.2

Currently focused on stabilizing the foundation before implementing advanced simulation systems.
